## Overview

An implementation of the AlphaZero algorithm applied to Connect 4. Combines Monte Carlo Tree Search (MCTS) with deep neural networks to learn game strategy entirely through self-play, without any human-provided heuristics.

**Tech Stack:** Python, PyTorch
